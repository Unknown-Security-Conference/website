AWS Advanced Offensive Techniques, what defenders need to know.

Las plataformas cloud como Amazon Web Services (AWS) son fundamentales para muchas infraestructuras críticas, convirtiéndolas en objetivos frecuentes para atacantes.

Esta charla explora los vectores de ataque más relevantes utilizados por actores maliciosos para comprometer entornos AWS, mediante la emulación de amenazas reales.

Desde el acceso inicial hasta técnicas avanzadas de persistencia, se analizarán tácticas y procedimientos (TTPs) alineados con el marco MITRE ATT&CK, incluyendo explotación de credenciales, manipulación de IAM y evasión de CloudTrail.

Se demostrarán ataques en vivo, estrategias defensivas y el uso de herramientas nativas de AWS y de terceros para mejorar la capacidad de detección y respuesta.

Una sesión práctica y altamente técnica con apenas dos diapositivas, enfocada en proporcionar experiencia real sobre cómo proteger infraestructuras en la nube ante amenazas sofisticadas.
